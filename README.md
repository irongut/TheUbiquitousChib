Releases are published on Nexus:

- [The Ubiquitous Chib - Dogtown](https://www.nexusmods.com/cyberpunk2077/mods/20913)
- [The Ubiquitous Chib - North Badlands](https://www.nexusmods.com/cyberpunk2077/mods/20995)
