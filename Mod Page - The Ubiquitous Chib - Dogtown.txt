Mod Page: The Ubiquitous Chib - Dogtown

Category = Vendors
Name = The Ubiquitous Chib - Dogtown
Language = n/a
Version = 1.0.0
Author = Irongut
Overview = Adds a vendor to Longshore Stacks in Dogtown who sells all melee weapons, mods and performance enhancements.
Description = 
Are you down on your luck or on the run?
Have you fallen off Skid Row and ended up in Dogtown?
Did you pawn your Katana for Black Lace?
And your favourite Tomahawk got stuck in a Scav skull this morning?

Then come on down to the Sky Gym in Longshore Stacks where our staff will help you find the melee weapon of your dreams!
Our stealth drones deliver new stock all the time so check back regularly! (1 hour refresh)

Also Available: The Ubiquitous Chib - Watson


Files
Upcoming Merc
Level appropriate melee weapons like a vanilla vendor; Rare and Epic Blade, Melee, Thrown and Blunt mods; and a selection of combat performance enhancements. Designed for use from the start of Phantom Liberty to respect gameplay balance.

Legend of NC
Legendary (Tier 5) melee weapons; Legendary (Tier 5) Blade, Melee, Thrown and Blunt mods; and a selection of combat performance enhancements. Designed for use at level 60 or if you don't care about gameplay balance.

INSTALL ONE.
Both versions overwrite each other so you can swap between them as you progress through the game if you want to.

Requirements
ArchiveXL
Codeware
RED4ext
redscript
TweakXL
VendorsXL

How to Install
Install all the requirements.
Download EITHER the "Upcoming Merc" (level appropriate) OR the "Legend of NC" (all legendary) archive from the files section.
Unpack the zip file into the "Cyberpunk 2077" folder so that the "TheUbiquitousChib_Dogtown.yaml" file ends up in the "Cyberpunk 2077\r6\tweaks" folder.

How to Uninstall
Delete the "TheUbiquitousChib_Dogtown.yaml" file from the "Cyberpunk 2077\r6\tweaks" folder.
